Chess Analytics - Analysis of games played on Free Internet Chess Game servers.
==============================

### Report
You can download and view in your browser the visual report with written analysis _Report.html_ file. Alternatively the report is available [here](http://rpubs.com/nktbinh214/296498)

#### To replicate/run the analysis:
- Download the data file: _rawdata/ficsgamesdb_201701_CvH.pgn_
- Download the 8 required R files: _common.R_, *00_installPackages.R*, *01_PGN_to_CSV.R*, *02_Manipulation.R*, *03_rchess_package.R*
- Make sure all the above files are in the same directory
- If this is the first time running the pipeline, you will need to run 00_installPackages.R to install the required R packages
- Run Makefile.R (ensure the working directory is set to the above directory)

#### File descriptions:
- **figure/** - All plots and tables generated by the analysis are outputted in this directory
- **rawdata/** - All raw unprocessed inputs 
- **Makefile.R** - The main R file that runs all the scripts
- **00_installPackages.R** - Installs required packages
- **01_PGN_to_CSV.R** - Clean, process and convert the data
- **02_Manipulation.R** - Exploratory Data Analysis
- **03_rchess_package.R** - Analysis of Piece Movements using the rchess package
- **common.R** - Has common functionality needed for most scripts
- **1701_HVC.csv** - Processed, clean data
- **Report.Rmd**, **Report.html** - The R Markdown and HTML of the report version of this project
